## List

Your task is to build the list of cats below the random cat fact

Make a nice list at the bottom that will show cat name, breed and age.

## Styling
- You must use only bootstrap utility classes

## Technical Requirements
- The list must be in the same component - app.component.html
- Do not create any new component
- Make sure to create Model and Service for it
- Use local json stub file for it located in public/cats.json file
